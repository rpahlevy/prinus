primaBase

Merupakan paket software yang berfungsi untuk menerima data periodik dari primaGateway dan fungsi manajemen data Hujan dan Tinggi Muka Air serta data pendukung. primaBase melakukan penerimaan data dan manajemen data dasar sdan fungsi-fungsi untuk menjaga sistem telemetri berjalan baik.

Catatan. Software primaBase harus jalan pada salah satu komputer di kantor (Hidrologi).

Penampilan Utama

  • Software Aplikasi berbasis Web
  • Jalan dalam lingkungan LAN (private)
  • Menggunakan sebagian besar software Open Source
  • Memungkinkan diakses melalui Smartphone pada lingkungan kantor

LAN dan koneksi Internet

LAN (Local Area Network) sagar memungkinkan primaBase diakses oleh beberapa komputer/laptop pada LAN.

Wajib tersedia koneksi Internet 24 jam, diperlukan untuk dapat menerima data dari primaGateway.

Internet dengan bandwidth 1 MegaBytepersecond sudah cukup.

PC / Server

  • Prosesor dengan multicore atau single core
  • kecepatan clock 2.4GHz,
  • RAM / Memory 2GB,
  • Hardisk 250GB,
  • Ethernet 1Gbit
  • Speed: 8MHz
  • Sistem Operasi Ubuntu Server 16.04 LTS
  • Database Postgresql versi 9.6
  • Python 2.7
